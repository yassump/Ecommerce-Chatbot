AI Smart shopping Assistant
An AI-powered ecommerce assistant build using streamlit and Google Gemini API,deployed on AWS EC2.
This application helps users discover products,compare options and make better purchase decisions through conversational AI.
Features:
- Product recommendations based on budget
- Helps to compare different options
- suggest products based on user needs
- interactive chatbot interface
- Real-time responses using Generative AI
